# Aidan Smagh Personal Website

This is the source code for my personal website.
Live site: [https://aidan-smagh.github.io](https://aidan-smagh.github.io)

The site was built with React and deployed on Github Pages. Additionally, Redux was used to manage the state for the filtering as well as Tailwind for the css.

# Overview
The site highlights
- Who I am, what I'm up to, my career aspirations
- My experience
- My projects (filterable from the Hero section)

# Development
To download and run this repository locally:
```
npm install
npm run dev
```
